About Rate your Cup of Coffee! Description This is a simple web application for the module Web Engineering at the FHNW.
'I need Coffee' helps coffee-lovers to collect and rate the different coffee blends they have tasted.
Coffee-lovers are able to add new blends (coffee-mixes) to the database, edit or delete them and leave ratings & reviews.

The application is based on 3 Entities:

  • CoffeeMix
  • Bean
  • Rating

with following Relations:

  • CoffeeMix - Bean (n:n)
  • CoffeeMix - Raiting (1:n)

 Frontend This project was generated with Angular CLI version 13.3.3. As a UI component library we used Material Angular UI and as a e2e Testing Framework we implemented Cypress . Our code was formatted with the help of Prettier .

Setup Angular Application

npm i @angular/cli

Run

Run npm install .
Run ng serve for a dev server. Navigate to http://localhost:4200/.

Running e2e Tests

Run npx cypress open . Run the all or single tests in the cypress browser window.

Run Prettier

Run npx prettier --check . to check your code. Run npx prettier --write . to format your code.
