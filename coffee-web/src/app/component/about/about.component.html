About Rate your Cup of Coffee!

This is a sample application for the module Web Engineering at the FHNW. It shows a list of coffee blends (coffee mixes), allows to add and edit new blends and to review a single cup of coffee.

The application is based on 3 Entities:

  • CoffeeMix
  • Bean
  • Rating

with following Relations:

  • CoffeeMix - Bean (m:m)
  • CoffeeMix - Raiting (m:1)

Consult the READ ME' to run the application.
